# OBJ 1.2 Fencing & Bollards 
## Fence
- structure that encloses an area using panels or posts 
1. provides visual boundary/deterrent, shouldn't be violated by unauthorized personnel 
2. establishes physical barrier against unauthorized entry 
3. delays intruders, helps provide security personnel with longer window of time to react 
## Bollards
- robust, short vertical posts, steel/concrete, designed to manage or redirect vehicle traffic 
- used to keep vehicles away from buildings so if IED explodes, building/people are safe 
1. creates phys barrier, shields pedestrians, structures, assets from potential vehicle collisions 
2. clear visual reminder where vehicles not permitted
