# Door Locks OBJ 1.2
- phys security control that's designed to secure entryways by restricting/regulating access to a particular space or property 
## False Acceptance Rate (FAR)
- rate that the system auths a user as valid, even if person shouldn't have been granted access to the system 
## False Rejection Rate (FRR)
- occurs any time the biometrics system denies a user who should've been allowed access to the system 
## Equal Error Rate (EER)
- aka Crossover Error Rate (CER)
- uses measure of the effectiveness of given biometrics system to achieve a balance

# Access Badge Cloning OBJ 2.4
- copying the data from an RFID or NFC card/badge onto another card or device 
1. attacker can use handheld RFID/NFC reader to capture the data 
2. once captured: extract relevant auth credentials from the card 
3. use specialized writing tools transfer extracted data onto blank RFID/NFC card 
4. attacker has cloned access badge/device
