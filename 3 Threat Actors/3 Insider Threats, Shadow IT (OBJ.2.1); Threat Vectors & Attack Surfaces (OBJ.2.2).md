# Insider Threats (OBJ.2.1)
- threats that originate from within the org 
- intimate knowledge of the org's infrastructure 
- have varying levels of capabilities 
	- Data Theft
	- Sabotage
	- Misuse of Access Privileges 
**Driven By:**
1. Financial Gain, want profit from sale of sensitive org data to others 
2. Revenge, aiming to harm the org due to some kind of perceived wrong levied against the insider 
3. Carelessness/Lack of Awareness, lack of cybersecurity best practices by one of the org's users 
---
# Shadow IT (OBJ.2.1)
- the use of information technology systems, devices, software, applications, & services w/o explicit org approval
	- Use of Personal Devices for Work Purposes (BYOD)
	- Installation of Unapproved Software
	- Use of Cloud Services that Haven't been Approved by the org

# Threat Vectors & Attack Surfaces (OBJ.2.2)
### Threat Vector
- means or pathway by which an attacker can gain unauthorized access to a computer or network to deliver a malicious payload or carry out an unwanted action 
	- messages
	- images 
	- files
	- voice calls
	- removable devices 
	- unsecure networks 

**Messages**
- includes email threats, simple message service (SMS) text messaging, or other forms of instant messaging 

**Images**
- image-based threat vectors involve the embedding of malicious code inside of an image file by the threat actor 

**Files**
- file-based threat vectors involve the use of malicious files to deliver a cyber threat 

**Voice Calls**
- voice call-based threat vectors involve the use of voice calls to trick victims into revealing their sensitive information 

**Removable Devices**
- Removable device threat vectors refer to threats delivered via removable devices such as USB 

**Social Engineering**

**Unsecure Networks**
- refer to the lack of appropriate security measures to protect networks 

**BlueBorne**
- set of vulnerabilities in Bluetooth technology that can allow an attacker to take over devices or spread malware 

**BlueSmack**
- type of Denial of Service attack that targets Bluetooth-enabled devices by sending a specially crafted Logical Link Control & Adaptation Protocol packet to a target device 
### Attack Surface 
- Encompasses all the various points where an unauthorized user can try to enter data to or extract data from an environment 
	- Restricting Access to network 
	- Removing Unnecessary Software
	- Disabling Unused Protocols
